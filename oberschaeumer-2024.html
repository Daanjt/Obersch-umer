Datei muss gehostet werden

Du hast die HTML direkt geöffnet (file://). Browser blockieren in diesem Modus alle Verbindungen. Du musst die Datei erst online stellen — geht in 1 Minute, gratis:

  1. Öffne tiiny.host
  2. HTML-Datei reinziehen
  3. Du bekommst eine URL — die statt der lokalen Datei verwenden

Mit Supabase verbinden

Einmaliges Setup. In Supabase:

  1. Projekt öffnen (oder neues erstellen)
  2. Linkes Menü: SQL Editor → „New query" → folgendes SQL einfügen und Run:
CREATE TABLE IF NOT EXISTS oberschaeumer_poll (
  id TEXT PRIMARY KEY,
  data JSONB DEFAULT '{}'::jsonb
);
ALTER TABLE oberschaeumer_poll DISABLE ROW LEVEL SECURITY;
